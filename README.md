# Understanding Mediation

This repository contains an interactive tool for visualizing and understanding statistical mediation analysis.

## Live Website
The tool is hosted and accessible at:
**[https://saulpascualdiaz.github.io/understanding_mediation/](https://saulpascualdiaz.github.io/understanding_mediation/)**

---

## Overview
This project illustrates the relationship between an independent variable (X), a mediator (M), and a dependent variable (Y). It allows users to explore:
* **Path a:** Effect of X on M.
* **Path b:** Effect of M on Y controlling for X.
* **Path c':** Direct effect of X on Y.
* **Indirect Effect:** The product of paths a and b ($a \times b$).
